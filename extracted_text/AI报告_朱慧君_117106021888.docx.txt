人工智能在金融风控和信用评级中的应用综述



摘要

人工智能产业作为一个新兴领域，在世界范围内还处于起步阶段，我国人工智能领域的研究及应用水平与国外发达国家相比不分上下，可谓平分秋色。未来，人工智能产业的竞争也会伴随不断增长变化的需求而演化。企业也会在不断满足并提升社会大众丰富多彩的生活品质而进步。本文将对人工智能在金融风控和信用评级中的应用做一个简单的介绍。

关键字：人工智能 金融风控 新用评级 神经网络 支持向量机



引言

人工智能（Artificial Intelligence,AI），是计算机科学、控制论、信息论、神经生理学、心理学、语言学等多学科相互渗透而发展起来的一门综合性学科。人工智能研究的领域包括机器人、语音识别、图像识别、自然语言处理和专家系统等。从字面上看，人工智能就是使计算机具有类似于人的智能，这也是人工智能的根本目标。因此，它不仅受学术界瞩目，也被产业界广泛关注。特别是AlphaGo的光荣战绩也让人工智能的潜力在各界激起涟漪。



一、国内外发展现状

1.1 人工智能发展的国外态势

国际人工智能技术与应用的发展主要由几个主要的互联网公司进行推动。

谷歌公司：基于人工神经网络进行研究，开发出了一种新型的学习算法，能够模仿人脑学习和记忆的原理，并且具有对数据进行归纳整理以及进一步预测推断的能力。在自动驾驶领域，2016年3月，谷歌公司研制的自动驾驶汽车成功完成总计约150万英里的路测。而在语音识别和图像识别领域，其开发算法的精准度也在不断突破。

Facebook公司：致力于通过人工智能技术进行用户数据分析以提升用户体验，主要布局基于用户信息和关系的人工智能算法演进，同时关注于图像识别和自然语言处理，其脸部识别算法的准确度已达到97%。

IBM公司：依托自身计算基础重点部署大规模计算能力，研制能模拟人类大脑的芯片，该芯片有一百万个神经元，2.56 亿个神经突触和4096个神经突触内核。

除此之外，近年国际上已经陆续开展了人脑模拟和研究的计划，一方面通过模拟人脑促进医学进展，另一方面通过研究人脑的学习机制助推学习算法、人机交互以及神经计算的进一步发展。

1.2 人工智能国内发展现状

目前我国的互联网企业中阿里巴巴和腾讯把主要的精力放在了横向圈地动作上，以求获得更多的连接面，百度则持续以技术为核心，聚焦纵向，进行人工智能和深度学习领域的布局。百度云结合深度学习技术提供的人脸识别及检索技术，使用类似人脑思维的方式去识别、搜索图片中的物体和其他内容。语音识别系统可以在嘈杂环境下实现将近 81%的辨识准确率。同时百度已经建成名为“百度大脑”的大规模深度神经网络，目前可以理解分析200亿个参数，达到了两到三岁儿童的智力水平。纵观我国人工智能技术与产业发展，中科院自动化所、清华大学、百度、科大讯飞等科研机构和公司均已经开展深度学习理论算法、建模等方面的研究，有关人脑网络结构与认知结构的研究也开始开展。国防科技大学、中航科工、百度、科大讯飞等企业的部分人工智能相关技术开始实现产品化应用。

人工智能理论和技术日益成熟，几乎各个领域的技术发展都涉及人工智能技术，其应用影响特别是在金融领域不断扩大。本文将从人工智能在金融风控和信用评级中的应用情况来浅谈人工智能在金融领域的发展现状。

二、人工智能在风控中的应用

汇总整个互联网金融中的风险，其实存在两个方面，一是道德风险，二是经营性风险。面对 2016年不断有“跑路”等负面消息萦绕的互联网金融，去伪存真或成为首要任务。监管层希望通过资金的有效监控，将企业资金与个人用户之间的资金进行分离，规避风险。然而人力毕竟有限，不可能时刻紧盯住所有互联网金融机构，这时引入人工智能监管就十分必要。

2.1风险控制体系发展中的经验理论体系

“经验”概念，实际上是一种个性化大数据的统计方法。一个人在某个行业根据经历所形成的一套观察事物和分析事物的思维方式。这个思维方式表现为以下几点。

（1）特定的指标体系。比如一个多年经营服装店的人，会根据装修、服饰和客流量这三个指标来衡量商店的经营能力，就会形成基于这三个指标来衡量经营能力的统计模型。

（2）特定的计算方式。除了指标体系，我们的思维方式中还包含着计算公式。假定装修（X1）会导致客人提升愿意掏钱购物的标准，服饰风格定价为导致客人实际消费增加，那么X1和X2联合作用时，甚至再加上服务水平提升X3时，单客价Y会变成怎样？

（3）特定的统计学标准。除此之外还要有行业或者现象的可比数据，如采取古典式门店装修，就需要访问多家采取同样装修的餐馆，看看他们的平均客单价，经过简单人脑中的计算，再看到同样的装修时，自然会反馈出相应的客单价来。

（4）特定的限制条件。装修和服饰风格的提升可以带来客单价的提升，但客单价的提升又会导致客流量相应的变化，于是服饰的定价就被限制在一个既能增加效益，又不会减少客流量的范围内，这个常识会形成经验中的特定的限制条件。

（5）特定的逻辑。当最优解不唯一时，经验就会进入到下一个层面的优化过程中，即逻辑决策优化过程。以一个学生买家为例，他首先会建立一个挑选的指标体系，如选择布料舒适的衣饰/选择自己喜欢的颜色/选择价钱在自己消费能力范围内/选择服务态度热情的；为了按照自己的指标体系挑选合适的上架，他会根据经验得出一个选择公式。

2.2传统风控模式的体系介绍

传统风控模式，采取的一直是以人工经验为主的专家意见模式，信审工具获取结论的过程中依赖人工经验判断，所以信贷评审人员的水平和悟性无形中决定了项目评审风险控制的能力，而有经验高水平且道德良好的信审人员的获取及培养，对于任何一家金融机构来说都是小概率事件，这使得经营贷的风控能力很难适应大规模和高效审批的要求。

此外，在个人小额贷中，虽然采取了大数据后台以及信贷工厂模式，在一定程度上解决了批量化和标准化问题，但用什么样的初始模型来进行机器审核却一直是困扰风控的问题。设计初始模型的业务人员，往往不懂数据统计方法，凭经验设计的风控模型是否能得到验证，就需要采取试投放后获得信贷结果数据再统计的方式才能进行验证。一般一类模型都需要经历几十万个信贷结果数据才能得到比较好的校正，但这个过程中需要信贷机构付出的学费，有可能十分昂贵。信贷行业一直用固定的套路和固定的信审模式来进行项目风控，因此老业务人员的经验一直是传统风控模型的核心。新的业务品种开发，往往也是在常年经营经验基础上的延伸，直到互联网带来了场景的大爆发，使得新产品创新的力度要求空前高涨，靠老经验已经很难适应新技术时代新产品模型更新的速度要求，而大数据的发展，也使得过去依赖经验判断的很多事项，有了后台数据的直接依据，这两方面的发展，都推进着信贷风控模型的创新。

2.3金融风控向人工智能方向转换

通过运用人工智能的深度学习系统，能够有足够多的数据供其进行学习，并不断完善甚至能够超过人类的知识回答能力，尤其在风险管理与交易这种对复杂数据的处理方面，人工智能的应用将大幅降低人力成本并提升金融风控及业务处理能力。

传统风控应用人工智能的第一步就是需要从经验中提取逻辑，而风控模型创新，并非是现在流行的大数据机器建模。这种场景下的原始模型往往是借用他人或由有经验的人员创建出来的。

如做一个简单的二手车贷款，模拟该经验形成的过程如下：

(1)首先形成一个挑选车主的指标体系：如必须要有车，车辆要有正常行驶的能力，车主有用款需求，车辆手续必须齐全等，以此哎作为衡量收车的基本门槛。

(2)收车的经验中会有特定的计算方式。如说服车主成功来贷款的可能性=车主用款的急迫性×利率的比较优势×我方放款快速性优势×我方放款充足性，由此得到一个可能说服车主来抵车的成功可能性指数。

(3)收车经验中的统计学标准。收车中成功率就是个统计学标准，当说服车主的可能性指数对应的成功可能性，是一个动态数据标准，存在以下的判定过程。根据每次收车成功的指数计算曲线，自动得出一个动态的平均线位置，并与平均值进行比对，从而得到一个此次指数分布点位置，与平均值比对后得到此次成功收车的可能性，因而决定收车的基本预期判断。

(4)收车中的限制性条件。决定了收车预期判断之后，经验中会自动链接收车的限制性条件，如某些品种和年限的车辆，很难处置或批贷率很低，因此会在经验中列入限制性条件。

(5)收车的特定逻辑：筛选圈定了可能成功贷款的车主后，自然会形成一个智能逻辑决策树模型：收车成功性=借款人职业批贷率×特定品种车批贷率×车辆年限有效率×借款人身份所在地批贷率×(1+其他因子调整)×特定的谈判条件。

分值落在合理区间内时，即可形成收车的决策，而落在排除区时，经验就会提示收车人放弃或考虑用非常规方式去做争取。

同样的经验也发生在信审环节上，车贷的风控审批之前，一般要经历手续审核、车况审核、车价评估等环节，风险控制的主要任务是通过上述环节得到的资料，来综合判断是否批贷，以及批贷的额度、周期和利率等。信审决策树的目的就是对这个过程进行建模，对各项指标进行打分，并按照一定规则计算最终得分来进行最终判断。

目前银行中拟合经验的主要模式就是规则引擎系统，其基本原理为将传统信贷人员的工作经验以规则库模式放入系统，由产品设计人员选取规则来辅助信审流程。



三、人工智能在信用评级中的应用

公司的资信等级反映的是其偿债能力和风险的大小，正确预测上市公司的信用风险状况，对于投资者做出最佳选择具有重要意义。上市公司的资信状况也是上市公司的社会形象、融资能力的表现，因而也是上市公司竞争力和发展潜力的反映。

人工智能技术诸如神经网络、专家系统、分类树等方法也逐步被运用到信用评级中去。传统的企业资信评级方法存在着如主观性因素太大以及权重确定困难等缺陷。人工智能技术中的神经网络方法具有很强的非线性映射能力和学习能力，但存在局部极小问题以及解释能力差等问题，支持向量机则在有限样本的情况下，展示了它优秀的非线性映射能力和泛化能力。以下将神经网络和支持向量机在企业信用评级中的应用作了对比，并分析它们的原理。

3.1 BP神经网络

人工神经网络（Artificial Neural Network）是由大量神经元互联而成的数学模型，通过模拟人脑的生理结构来模拟人工智能，具有存储和应用经验知识的特性。它与人脑的相似之处概括为两个方面：一是通过学习训练，利用神经网络从外部输入中获取知识；二是使用内部神经元（突触权值）存储获取的信息。神经网络提供了输入和输出之间的灵活映射，可以用于函数逼近和模式识别。

众多神经网络中，误差反向传播（Error Back Propagation）的BP神经网络由于其良好的逼近能力和成熟的训练方法而得到了最为广泛的应用。BP网络是一种前馈型网络，由一个输入层、若干隐含层和一个输出层构成。如果输入层、隐含层和输出层的单元个数分别为n,q,m，则3层网络可以表示为BP（n,q,m），利用该网络可以实现n维输入向量Xn={X1,……,Xn}T向m维输出向量Ym={Y1,……,Ym}T的非线性映射。BP网络中，位于同一层的单元之间没有连接，各层的单元只能向高层的单元输出信号。BP网络中的参数有从输入层第i单元到隐含层第j单元的权重Wij，隐含层第j单元到输出层第k单元的权重Wjk，隐含层第j单元的激活阈值θj，输出层第k单元的激活阈值θk。BP网络的核心是通过向前传递计算网络的输出值，并与期望输出值比较，将误差反向传播，通过不断调整网络中的参数最小化网络输出与期望输出之间的误差，以达到逐步逼近期望映射关系的目的。

BP的训练过程可分为两部分：第一部分是信息的正向传播过程，在此过程中，训练样本信息首先向前传播到隐含单元上，再经过激活函数f(u)的作用，得到隐含层的输出信息Hjp，隐含层的输出信息传到输出层，可得到最终输出结果；第二部分是误差的反向传播过程。如果网络输出与期望输出之间存在误差，则将误差反向传播以调节网络权重和阈值，直到网络的测试效果达到最佳且稳定。



3.2支持向量机

在统计学习理论的基础上发展出了一种新的通用学习方法——支持向量机（Support Vector Machine或SVM）， 它已初步表现出很多优于已有方法的性能。一些学者认为，SVM 正在成为继神经网络研究之后新的研究热点，并将有力地推动机器学习理论和技术的发展。SVM是从线性可分情况下的最优分类面发展而来的，基本思想可用图示的两维情况说明。图中，实心点和空心点代表两类样本，H为分类线，H1、H2分别为过各类中离分类线最近的样本且平行于分类线的直线，它们之间的距离叫做分类间隔(margin)。所谓最优分类线就是要求分类线不但能将两类正确分开（训练错误率为0），而且使分类间隔最大。分类线方程x·w + b = 0，我们可以对它进行归一化，使得线性可分的样本集( xi , yi )，i=1,……,n,x∈Rd,y∈{+1,-1},满足yi[ ( w·xi ) + b ] - 1 ≥ 0，i=1,……,n，此时分类间隔等于2/||w||，使间隔最大等价于使||w||2最小，满足上式条件且使1/2||w||2最小的分类面就叫做最优分类面，H1，H2上的训练样本点就称作支持向量。

以上是二维情形，推广到多维，支持向量机就是首先通过用内积函数定义的非线性变换将输入空间变换到一个高维空间，在这个空间中求（广义）最优分类面。SVM 分类函数形式上类似于一个神经网络，输出是中间节点的线性组合，每个中间节点对应一个支持向量，如图所示。

SVM中不同的内积核函数将形成不同的算法，目前研究最多的核函数主要有三类：

1）多项式核函数

K ( x , xi ) = [ ( x·x i ) + 1 ]d ;

2）径向基函数核函数

K ( x , xi ) = K ( | x - xi | );

3）两层神经网络的核函数

K ( x , xi ) = S ( v ( x , xi ) + c ).

3.3 ANN与SVM应用的比较

在样本量相同的情况下，支持向量机(SVM)的评估结果优于神经网络(ANN)的评估结果。随着样本量的增加，ANN和SVM的评估准确性均有所提高在样本量有限的情况下，支持向量机评估准确性优于BP神经网络，具有良好的应用前景。在进行企业资信评级时，如果行业间存在较大的差距，则可对不同行业进行具体建模，以提高评判的准确性。



四、人工智能其它应用

4.1 自动驾驶

自动驾驶及其相关领域在国外起步较早，也有了一定的研究基础。汽车制造商如宝马公司在2009年就启动对于自动驾驶汽车的研发，在2011年其原型车就已经完成了高速公路测试。互联网公司如谷歌从2005年开始研究自动驾驶技术，经过多年研发技术已经相对成熟，2015推出完全通过软件和传感器实现自动驾驶的汽车。在国内，百度是主要进行自动驾驶研究的公司，百度基于地图服务、深度学习技术、图像识别技术，计算视觉等方面的积累优势，与宝马公司共同合作研发智能自动驾驶汽车，并在2015年已完成实际道路的上路测试。最新消息指出，中国长安汽车和 “美国版滴滴打车”，近日相继获得美国政府许可，得以在加州公共道路上测试自己的无人驾驶车。不远的将来，我们的出行很可能经历极大的转变。

4.2 自然语言处理

从2006年开始，研究者们开始利用深层神经网络在大规模无标注语言源上进行无监督学习，在形式上把每个词表示成一个固定维数的向量，当作词的本身特征，在此特征基础上，进行架构设计、分词标注、分块、命名实体识别、训练网络及语义角色标注等操作。以这样的方法进行自然语言处理可以实现高计算速度的大数据处理，而如果使用多任务模式进行计算还能进一步提升系统的计算和处理速度。

4.3 图像识别

目前，百度采用深度卷积神经网络作为模型，并在构建的近1亿图片样本上进行端到端的学习，已经可以识别超过5万的标签，在已知公开报道的同级别模型中，该模型性能远超公开的性能指标。除此之外，目前百度还正在进行图像的自然语言描述方面研究。利用百度的深度学习能力，结合卷积神经网络和递归神经网络技术百度成功地实现将图像内容生成自然语言的描述性句子或段落，从而在高层语义层面建立了图像和自然语言之间的桥梁。

4.4 智能机器人

智能机器人的研制始于20世纪90年代末，并迅速开展相关研究工作。2005年，意大利比萨大学研制出表情机器人FACE，并将其应用于自闭症辅助治疗等领域。



五、结论与展望

人工智能产业作为一个新兴领域，在世界范围内还处于起步阶段，我国人工智能领域的研究及应用水平与国外发达国家相比不分上下，可谓平分秋色。未来，人工智能产业的竞争也会伴随不断增长变化的需求而演化。企业也会在不断满足并提升社会大众丰富多彩的生活品质而进步。
