摘要:

随着近几年计算机学科的发展，人工智能等技术与搜索引擎的结合催生了智能搜索引擎。智能搜索引擎以其独有的智能化、人性化和学习性等特征，受到广大网络用户的欢迎。但是信息的爆炸式增长，使得搜索引擎搜索结果准确度不高、用户交互体验低等缺点也逐渐暴露出来。 

关键词检索是大多数搜索引擎的主要检索手段，然而多数情况下，检索效率不尽如人意。本文主要围绕智能搜索引擎的搜索算法展开，介绍了一种PISE爬虫算法，这种智能搜索算法基于“个性化搜索”的技术，使用户的搜索更加精确。另外还介绍了一种个性化PageRank算法，可以对用户的检索要求进行智能理解，提供个性化的服务，并且拥有更加强大的搜索能力。



关键词： 智能搜索，智能搜索引擎，Agent，PISE爬虫算法，个性化PageRank算法

研究目的和意义

现今，随着网络信息的爆炸式增长，网络上虽然有庞大的信息资源，但是其固有的开放性、动态性、异构性、资源分布的分散性等特点，使得人们很难快速准确的检索出自己想要的信息，为了简化用户对信息资源的访问和获取，出现了专门帮助人们检索网上信息资源的检索工具——搜索引擎(Search Engine)。

目前的搜索引擎还存在的一些问题，诸如检索速度慢，无效链接多，重复或不相关信息多，搜索结果不具有个性化，不能因人而异等问题。智能搜索引擎，又称第三代搜索引擎，是新一代智能技术应用在搜索引擎上所得到的更加高效的搜索引擎技术。它的出现弥补了传统搜索引擎在用户使用时的一些不足，并且依靠其自身的智能化给用户带来许多新的更加人性化的功能。 

国内外研究现状和发展水平

搜索引擎，尤其是智能搜索引擎以其强大的功能受到越来越广泛的应用，也越来越多地受到国内外学者的重视，这也使得众多学者从各个角度对其进行了研究。

2.1 国外研究现状

在实践方面，国外开发的搜索引擎比较多，搜索引擎的鼻祖是加拿大麦吉尔大学（University of Mc Gill）计算机学院的三名学生 Alan Emtage、Peter Deutsch、Bill Wheelan 开发出的 Archie 软件，而后的 Excite、Yahoo、Lycos、Google 等搜索引擎也纷纷在技术上推陈出新，不断拓展搜索引擎的功能。智能搜索引擎的出现缘于人工智能、知识发现等技术与搜索引擎技术的结合，因此国外学者对智能搜索引擎的研究多从技术角度出发，介绍新的搜索算法、研究性的搜索技术来提高搜索引擎的功能。主要研究有： 

（1）搜索引擎算法研究。

Hersovici M， Heydon A， Mitzenmacher M 等人提出智能搜索引擎的网络蜘蛛搜索算法——鲨鱼搜索算法，能够避免搜索与主题不相关的网页，从而提高检索命中率；Filippo Menczer， Gautam Pant， Padmini Srinivasan 等人论述了智能搜索引擎中的网络蜘蛛所使用的深度优先算法、PageRank 等搜索方法。

（2）搜索引擎技术研究。

Andrew Mc Callum， Kamal Nigam， Jason Rennie 等人介绍了一种新的爬行虫——Net Spide，因其具备巩固学习机制可以避免在搜索过程中搜索与主题无关的网页信息，从而被大量应用在网络智能搜索引擎中；Marat Kanteev， Igor Minakov， George Rzevski 等人介绍了一种使用语义网理解技术构建的多 Agent 元搜索引擎，并论述了它在网页分析、分类、个人咨询等方面的应用。 

国内研究现状

中文汉字的复杂性使得中文搜索引擎的发展与国外相比在技术方面比较落后。但是随着Google登录中国、百度公司的建立、北大天网的创立等，搜索引擎很快成为国内网络时代第三次发展浪潮的领军者，对中文搜索引擎的研究状况也随之发生了改变。国内计算机与软件方面的专家学者在智能搜索引擎技术方面进行了广泛的探讨，主要研究有： 

智能搜索引擎技术研究

刘敏提出了“以网对网”的提高搜索引擎智能水平的思路，介绍了 Internet 信息检索策略以及其相较于传统搜索引擎的特点。皮鹏，张国印提出一种采用分布式 Agent 技术，以词典作为信息匹配和交互的媒体，能够很好地对用户的兴趣进行自适应学习，对用户提供个性化的信息检索服务的专题型搜索的智能元搜索引擎方案，并给出了该方案框架结构和具体算法的描述。 

智能搜索引擎系统研究

韦鲁玉，丁华福从现有搜索问题入手，开发设计了一个基于 Agent 的个性化智能信息检索系统模型，该模型由用户信息检索个性 Agent、信息搜索 Agent 和信息过滤 Agent 三个模块构成，对三个模块中的关键技术进行研究后，结果表明该平台可减少搜索范围，加快搜索速度。

4.1 基于PISE爬虫算法的智能搜索

PISE(Personal Intelligent Search Engine)就是基于“个性化搜索”的技术，采用逐步求精的策略来实现个性化，同时实现了实时性。它的个性化体现在搜索引擎返回的信息是符合用户的个人想法的，而不是仅仅只与用户输入的关键字相匹配；实时性体现在它不是从本地服务器上存储的索引数据库中查找用户需要的信息，而是实时地到网上寻找最新信息，返回给用户。适应于这些特点，PISE的爬行虫在网址的初始化和爬行策略上具有一定的特色。

爬行虫的初始URL

用户使用PISE进行搜索时，首先输入要搜索的关键字，爬行虫接收到关键字后，到用户库中查询，看是否有关键字字段与用户的关键字相匹配，如果有则用相应的记录中感兴趣的网址字段的值作为初始的URL，否则用关键字字段为空的相应记录的感兴趣的网址字段的值(即用户经常浏览的网址)以及其他用户的对应统一关键字的感兴趣网址作为初始的URL。

每一次搜索都会向用户返回一些相关网址，服务器程序根据用户点击的次序及浏览的时间计算出兴趣度，把用户兴趣度到达一定数值的网址更新到用户库，同时存入查询的关键字，为下一次查询提供初始URL。并且更改库中网址的兴趣度，当兴趣度降低到一定值时就删除此记录。这样，虽然第一次查询可能返回结果不多也不太精确，但因为每一次新的查询都是在以前查询基础上进行的，初始URL都更接近用户的意图，查询的结果会越来越精确。

爬行策略

我们可以把网站按内容分为不同的类别。与我们关键字相匹配的只能是某一个类别，所以我们要查找的目的就变为要查找某一类别的网站，然后再从此网站中找出相应的网页。根据以上分析可以制定出PISE的爬行策略，如果某一个网页中没有与关键词匹配的文本，则可以判断出此网页的网站不属于我们要查找的类别，因此放弃内部链接，只把外部链接放入查询队列。另外，把相关锚点对应的链接网址放入优先队列，让这些网址对应的网页得到优先处理，以尽快给用户返回结果。

按照链接与关键词的可能匹配程度我们将爬行队列分为四级：最优队列、次优队列、等待队列和完成队列。如果某一网页中没有与关键词匹配的文本，则将此网页的外部链接放入次优队列；如果有与关键词匹配的文本，将匹配锚点的内部链接以及外部链接放入最优队列，不匹配锚点的外部链接放入次优队列，内部链接放入等待队列；已经处理完的链接放入完成队列。

网页相关度计算

为了提示用户相关性大小，返回相关度是很必要的。根据对相关性贡献的大小，我们分几个层次为网页的相关性赋值：

1)如果网页的title与关键字匹配，给此网页相关度加5。

2)如果网页文本中的小标题与关键字匹配，给此网页相关度加4；其他文本与关键字匹配，给此网页相关度加2。

网页文本相关度加权的方法：如果关键字在小标题中出现n1次，在普通文本中出现n2次，则总的相关度加权为：(4×n1+ 2× n2) /(n1+ n2)。

4.3 基于个性化PageRank算法的智能搜索

传统PageRank算法是根据网页的相互引用次数来评价网页的重要性，并以此在搜索结果中进行排序。其核心算法是，先给所有页面一个评分初值，然后PageRank值按照被引用次数在各个页面之间进行“流动”，最终达到平衡，导致被引用次数多的网页得到较高评分。这一技术初步保证了相对有用的信息会优先返回给用户。然而这种算法缺失了页面的分类信息和个性化信息。

如果要按照用户的要求、喜好和历史浏览记录来进行页面排序，可以使用和PageRank类似的技术，称之为个性化PageRank评分。该评分用来反映页面与用户个性化需求的匹配程度。可以参考传统PageRank的计算方法，以用户需求为中心，进行有限次PageRank值的流动，从而得到个性化PageRank值。将用户作为PageRank值的分发源，例如用户具有初始PageRank为1，其他网页为0，根据网页在用户历史记录中被浏览的时间和次数给出PageRank的传递方向和传递阻尼，经过有限次的传递(不需要达到平衡)，得到相关网页的个性化PageRank值。这样，用户浏览次数较多或着最近浏览的网页及其相关网页即可得到较高的评分，从而优先显示给用户。个性化PageRank的计算需要详尽的用户浏览记录，并且进行实时计算。

智能搜索在搜索引擎方面的应用

智能搜索引擎(Intelligent Search Engine，ISE)是运用人工智能（AI）、数据挖掘(Data Mining，DM)、自然语言理解等现代技术，将传统的基于关键词的检索提高到基于知识或概念的检索，在服务器端记录用户个人资料操作习惯，以知识逻辑推理机制明确用户检索目的，通过搜索器进行智能化信息搜集，并且过滤、推送信息至用户的系统。

而多 Agent 智能搜索引擎（Multi-Agent Intelligent Search Engine，MAISE）是在智能搜索引擎的基础上综合运用了 MAS、主动搜索和元搜索技术相结合的搜索引擎。MAISE 其本身不提供大型的搜索信息预存数据库，而是根据用户需要将检索请求分发到其他实际的独立搜索引擎或主动搜索 Agent 上，对这些独立搜索引擎或主动搜索 Agent 所返回的结果在本地数据库进行分析处理后再将搜索结果返回给用户。

百度、网络商城与Google

百度

百度是最大的中文搜索引擎，它开发了基于搜索引擎服务为主的多种信息产品。百度能提供以网页为主的搜索服务，以贴吧为主的社区服务，提供以百度文库和百度知道为主的知识性服务，同时根据地域和行业搜集相关信息，提供垂直搜索服务。百度还提供娱乐类型的服务如：视频搜索和搜索服务。。

百度的检索框非常简易，没有复杂的界面。百度的简洁化界面符合用户的搜索的心理特点，也使百度迅速获得了用户的支持。用户根据自己的需要分别选择不同的专题进行检索。百度支持关键词检索、短语检索和部分自然语言检索。用户只需要输入要査询的信息的一些关键词，然后在百度一下就可以方便快捷地获得所需要的相关信息资源。

目前, Google搜索引擎已经可以支持MS Word、MS Excel、MS Powerpoint、Adobe PDF等文件格式。上文提及的PageRank算法使得Google搜索引擎获得了巨大的成功。

随着网络信息量的快速增长和网络用户的个性化、专业化要求日渐增强,传统搜索引擎暴露出了越来越多的不足。智能搜索引擎的提出,综合了当前最新的人工智能、计算机网络、数据库、数据挖掘、机器学习等技术,给出了更加优越、更加人性化的搜索引擎框架。智能搜索在搜索引擎方面将会得到更广泛的应用。
